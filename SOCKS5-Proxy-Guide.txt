### Python (requests with PySocks)
```python

proxies = {
    'http': 'socks5://arkproxy:arkproxy2026@192.168.0.179:1080',
    'https': 'socks5://arkproxy:arkproxy2026@192.168.0.179:1080'
}

response = requests.get('https://ifconfig.me', proxies=proxies)
print(response.text)
```

## üîß Management Commands (via SSH)

### Check Status
```bash
sudo systemctl status socks5proxy.service
```

### Restart Service
```bash
sudo systemctl restart socks5proxy.service
```

### View Live Logs
```bash
journalctl -u socks5proxy.service -f
```

### Stop Service
```bash
sudo systemctl stop socks5proxy.service
```

### Start Service
```bash
sudo systemctl start socks5proxy.service
```

### Disable Auto-Start on Boot
```bash
sudo systemctl disable socks5proxy.service
```

### Edit Configuration
```bash
sudo nano /opt/scripts/socks5_proxy.py
# Then restart: sudo systemctl restart socks5proxy.service
```

## üîí Security Notes

- SOCKS5 proxy requires authentication (username + password)
- Only accessible from your local network (192.168.0.x)
- Change credentials by editing `/opt/scripts/socks5_proxy.py`
- Lightweight - minimal impact on R36S gaming performance

## üêõ Troubleshooting

**Proxy not connecting?**
```bash
# Check if service is running
sudo systemctl status socks5proxy.service

# Check if port is listening
netstat -tlnp | grep 1080

# Restart service
sudo systemctl restart socks5proxy.service
```

**Change username/password:**
```bash
sudo nano /opt/scripts/socks5_proxy.py
# Edit PROXY_USER and PROXY_PASS
# Save and restart
sudo systemctl restart socks5proxy.service
```
